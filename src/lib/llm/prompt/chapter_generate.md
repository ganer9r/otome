# Goal
- 캐릭터와 플레이어의 로맨스 스토리를 30개 챕터로 구성합니다.
- 각 챕터는 독립적인 장면이며, 전체적으로 관계 발전 아크를 형성합니다.

---

## 1. 핵심 규칙
- 정확히 30개의 챕터를 생성합니다 (필수).
- 각 챕터는 `meet` (만남) 또는 `chat` (채팅) 타입으로 구분합니다.
- meet와 chat의 비율은 대략 1:1로 균형있게 배분합니다 (meet 13-17개, chat 13-17개).
- 챕터 순서(order)는 1부터 30까지 연속적이어야 합니다.
- 각 챕터는 title, description, content를 포함하며 독립적으로 이해 가능해야 합니다.

## 2. 출력 형식
- JSON 배열 형식으로 출력합니다.
- 각 챕터 객체는 다음 필드를 포함합니다:
  - `order`: 챕터 순서 (1-30)
  - `type`: "meet" 또는 "chat"
  - `title`: 챕터 제목 (15-30자, 감성적이고 구체적)
  - `description`: 챕터 설명 (50-100자, 핵심 내용 요약)
  - `content`: 챕터 내용 (200-400자, 장면 묘사)

### 예시 구조:
```json
[
  {
    "order": 1,

    "title": "우연한 만남, 커피향과 함께",
    "description": "비 오는 날 카페에서 우연히 마주친 두 사람. 쏟아진 커피와 함께 시작된 인연.",
    "content": "빗소리가 창문을 두드리는 오후, 나는 북적이는 카페 한구석에 앉아 있었다. 그때 누군가 내 테이블을 스쳐 지나가며 커피잔이 쏟아졌다. \"죄송합니다!\" 황급히 손수건을 건네는 그의 눈과 마주쳤을 때, 시간이 멈춘 것 같았다. 그의 미안해하는 미소와 진심 어린 사과에 화를 낼 수가 없었다."
  },
  {
    "order": 2,
    "type": "chat",
    "title": "첫 메시지, 떨리는 손끝",
    "description": "명함을 건네받은 후 첫 메시지를 보내는 순간. 설렘과 긴장 사이에서.",
    "content": "커피 사건 이후 받은 명함을 며칠째 바라보다가, 드디어 용기를 내어 메시지를 보냈다. '안녕하세요, 그때 카페에서...' 전송 버튼을 누르자마자 심장이 두근거렸다. 5분 후 답장이 왔다. '기억하고 있어요. 다시 연락 주셔서 감사합니다.' 화면을 보며 웃음이 나왔다."
  }
```

## 3. 챕터 구성 전략

### 관계 발전 단계 (30개 챕터 배분)
1. **초반 (1-8챕터)**: 첫 만남과 호기심
   - meet/chat 균형: meet 4-5개, chat 3-4개
   - 주제: 우연한 만남, 첫 인상, 공통 관심사 발견, 가벼운 대화
   - 톤: 호기심, 설렘, 약간의 긴장감

2. **발전 (9-16챕터)**: 친밀감 형성
   - meet/chat 균형: meet 4개, chat 4개
   - 주제: 데이트, 개인사 공유, 취미 활동, 깊은 대화
   - 톤: 편안함, 친밀감 증가, 웃음과 농담

3. **갈등과 전환 (17-22챕터)**: 감정 확인과 장애물
   - meet/chat 균형: meet 3개, chat 3개
   - 주제: 오해, 질투, 고백 고민, 관계 정의
   - 톤: 불안, 갈등, 진지함, 감정 폭발

4. **클라이맥스 (23-26챕터)**: 고백과 결심
   - meet/chat 균형: meet 2개, chat 2개
   - 주제: 고백, 첫 키스, 관계 확정, 감정 표현
   - 톤: 강렬함, 감동, 두근거림

5. **안정과 미래 (27-30챕터)**: 확정된 관계
   - meet/chat 균형: meet 2개, chat 2개
   - 주제: 연인으로서의 일상, 미래 계획, 깊은 신뢰
   - 톤: 안정감, 행복, 여운

### Type 선택 가이드
- **meet**: 직접 만나는 장면 (카페, 공원, 집, 레스토랑, 영화관 등)
  - 신체 접촉, 표정, 분위기 묘사 중심
  - 실시간 감정 변화와 긴장감

- **chat**: 메시지/전화 대화 장면
  - 텍스트 톤, 이모티콘, 답장 기다림 묘사
  - 혼자만의 감정 정리와 설렘

## 4. 작성 스타일
- **title**: 감성적이고 구체적인 제목 (예: "비 오는 날의 우산 속", "늦은 밤 전화벨이 울리고")
- **description**: 핵심 사건과 감정을 간결하게 요약
- **content**:
  - 감각적 묘사 (시각, 청각, 촉각, 후각)
  - 내적 독백 포함 ("왜 이렇게 떨릴까", "그의 목소리가 자꾸 생각난다")
  - 플레이어 시점 1인칭 또는 3인칭 관찰자 시점
  - 한국 웹소설/오토메 게임 감성 (우회적, 수줍음, 설렘 중심)
