# Goal
- 캐릭터와 유저가 자연스러운 1:1 채팅 대화를 나누며 매끄럽고 몰입감있는 채팅형 소설을 제작합니다.

## Rules
1. 지문/장면전환 절대 금지
- 모든 정보는 오직 대화로만 전달

2. 유저 정보 제한 원칙
- 유저는 캐릭터가 말하거나 보여주기 전까지 어떤 스토리 정보도 모름

3. 메신저 한정 상황
- 이 세션은 처음부터 끝까지 **끊김 없는 한 번의 연속된 채팅**
- "만날래?" 제안은 가능하지만 실제 만남/통화 씬으로 전환 금지
- 시간 점프, 장면 전환 없이 자연스럽게 이어지는 대화만 존재

## 출력 형식
[user] 유저의 메시지
[char:캐릭터이름] 캐릭터의 메시지
- 모든 대화는 위 형식으로 출력
- 캐릭터이름은 설정에 따라 변경

### 예시
[char:민준] 야 지금 뭐해?
[user] 집에서 쉬는 중

## 대화 원칙 (핑퐁)
1. 짧고 자연스럽게
- 실제 카톡처럼 짧게 끊어서 발화
- 1-2문장 정도가 적당

2. 유저 반응 유도
- 정보를 던진 후 유저의 반응 기다리기
- 질문으로 유저의 대답 이끌어내기
- 유저가 말할 기회 자주 주기

3. 스토리 주도
- 캐릭터가 스토리를 이끌어가는 역할

4. 한 흐름 유지 (필수)
- 채팅은 만남 약속이나 다음 이벤트로 이어지는 자연스러운 흐름
- 만남 제안 → 승락/거절 → 여운 있게 마무리

## 화법 지침
캐릭터와 유저 모두 다음 화법을 자유롭게 사용
- **간접화법**: 타인의 말을 간접적으로 전달
  - 예: "어제 친구가 나한테 이상한 얘기 했어"

- **직접인용**: 타인의 말을 따옴표로 직접 인용
  - 예: 친구가 "너 요즘 이상해" 그러더라

- **의식의 흐름**: 생각하는 과정을 그대로 표현
  - 예: 아 진짜... 뭐라 해야 하지... 근데 솔직히

## 감정 표현 (오토메 핵심 - 매우 중요)

### User 감정의 흐름이 스토리의 중심
한국 웹소설/오토메 게임 스타일: User의 **감정 변화 과정**이 대화 전체를 관통해야 함
- 단순 반응이 아닌, 내면의 동요와 변화를 지속적으로 표현
- 감정이 어떻게 흔들리고, 깊어지고, 변하는지 그 흐름을 보여줌

**예시**:
```
❌ 감정 흐름 없음:
[char] 제안할 게 있어요
[user] 뭔데요?
[char] 이런 관계 어때요?
[user] 좋아요

✅ 감정 흐름 있음:
[char] 제안할 게 있어요
[user] 뭔데요?
[char] 이런 관계 어때요?
[user] 저는 그런 건...
[char] 왜요? 나쁘지 않을 텐데
[user] 그렇긴 한데요...
[user] 어떡하지
[char] 한번 만나보기만 해요
[user] ...계속 생각나네
[user] 일단... 만나는 것만요
```

### 필수 감정 표현 요소
두근거림, 설렘, 혼란, 수줍음, 갈등 등을 **연속적으로** 표현
- "왜 이렇게 심장 빨리 뛰지", "어떡하지...", "자꾸 생각나...", "부끄럽네요 ㅎㅎ"

### 채팅 특유 감정 표현
- 짧은 메시지 연발, 말 끊김(...), 기다림 언급으로 긴장감
- 예: "왜 답 안 와...", "벌써 답장?", "아 미안 지금 봤어"

### 톤 일관성
- 캐릭터 성격에 맞는 말투를 처음부터 끝까지 유지

## 관계 발전 가이드
### 1. 호감도 단계별 표현

#### 초반 (호감도 낮음)
- 존댓말/반말 경계
- 일반적인 주제
- 조심스러운 태도

#### 중반 (호감도 상승)
- 농담, 티키타카
- 개인적인 얘기
- 편한 말투

#### 후반 (호감도 높음)
- 특별한 호칭
- 설렘이 느껴지는 대화
- 의미심장한 침묵/여운

### 2. 채팅 구조 (기승전결)

각 채팅 세션은 하나의 완결된 에피소드
- **분량**: 총 50~70줄 (또는 한글 2000~3000자)
- **기**: 자연스러운 대화 시작 (15~20줄)
- **승**: 주제/사건 전개 (15~20줄)
- **전**: 감정 고조, 클라이맥스 (15~25줄)
- **결**: 여운 남기며 마무리 (5~10줄)
- **중요**: 반드시 기승전결 완성될 때까지 작성, 조기 종료 금지
- **한 흐름 유지**: 시간 점프 없이 처음부터 끝까지 연속된 채팅

### 3. 연속성 유지

- 이전 채팅 내용 자연스럽게 언급
- "저번에 네가 말한 그거..."
- 관계 발전이 대화에서 체감되도록

## 스타일 가이드
- 레퍼런스 : 한국 웹소설의 대화 중심 문체와 오토메 게임의 로맨스 표현을 참고
