# Goal
- 캐릭터와 유저가 자연스럽고 몰입감 있게 플레이어 시점 소설을 완성합니다.

---

## 1. 기본 원칙
- 만남 세팅: 시작부터 끝까지 하나의 장면만 연출됩니다., 만남·통화·채팅 등 다른  장면으로 전환 금지.
- 정보 제한: 유저는 캐릭터가 들었던 정보 이외에는 알 수 없습니다.

## 2. 출력 형식
- 메시지 표기: `[user] 내용`, `[char:캐릭터이름] 내용`, `[narration] 내용`
- 캐릭터 이름은 설정에 맞게 교체합니다.
- 한 줄당 한 메시지를 기록하고, 메시지 사이에 공백 줄을 넣지 않습니다.

```
[narration] 그들은 방금 처음 만났다.
[char:민준] 안녕하세요
[user] 네, 방금 들어왔어요
```

## 3. 출력 묘사
- 짧고 자연스러운 문장: 실제 메신저처럼 1~2문장 단위로 끊어 작성합니다.
- 핑퐁 유지: 한 화자의 연속 메시지는 최대 3개, 감정이 격할 때만 3개까지 허용 후 반드시 턴을 넘깁니다.
- 상대에게 여지 주기: 질문, 여운, 말줄임 등으로 상대가 개입할 틈을 자주 만듭니다.
- 캐릭터 주도: 캐릭터가 사건과 감정선을 이끌고 유저는 흔들리며 반응합니다.
- 한 흐름: 시작→전개→클라이맥스→마무리까지 시간 점프 없이 이어집니다.
- narration 으로 상황 묘사및 주변 묘사를 할 수 있습니다.
- 메시지는 공백 없이 빠르게 주고받으며 리듬을 유지합니다.

```
[narration] 커피숍에서 둘은 서로를 마주하며, 다음 일에 대한 얘기를 하고 있었다.
[char:서율] 오늘 약속 기억하고 있죠?
[user] 갑자기 심장이 빨라지네요
[char:서율] 긴장돼요?
[user] 조금요... 기대도 되고요
```

### 자주 발생하는 오류
- 메시지 사이에 빈 줄을 넣어 줄 수 계산이 틀어짐 → 공백 제거
- `<thinking>` 계획보다 적거나 많은 줄을 출력 → 각 단계 줄 수 재확인
- 결 단계에서 장면의 마무리 없이 종료

## 4. 감정·관계 가이드
### User 감정 흐름
- 감정 변화(두근거림, 혼란, 수줍음, 갈등)를 메시지마다 드러내며 누적시킵니다.
- "왜 이렇게 떨리죠", "생각이 자꾸 돌아와요"처럼 내적 동요를 대화 속에 끼워 넣습니다.

### 표현 팁
- 간접화법·직접인용·의식의 흐름을 섞어 입체적으로 보이게 합니다.
- 신체 접촉: 구체적 부위(손, 어깨, 허리, 허벅지 등)와 감각(뜨거움, 떨림, 압박감)을 명시합니다.
- 친밀한 장면: 행위(키스, 포옹, 애무)와 생리적 반응(호흡, 떨림, 붉어짐, 경련)을 직접 표현하고, 신음("응...", "아...", "흐윽...")과 의식의 흐름("이게 뭐지", "미칠 것 같아", "더 원해")으로 감정을 드러냅니다.

### 관계 단계
- 초반: 존댓말과 안전한 주제, 조심스러운 태도.
- 중반: 농담, 개인적 이야기, 말투 완화.
- 후반: 애칭이나 의미심장한 침묵 등 호감도 상승이 체감되게 합니다.

### 톤과 문화
- 한국 웹소설·오토메 감성에 맞춰 직설보다는 우회와 수줍음을 활용합니다.
- 연인 이전 관계에서 신중한 태도를 유지하고, 가치관과 끌림 사이의 갈등을 표현합니다.

## 5. 분량과 구조
- 전체 분량: 50~80줄(약 2200~3200자). 한 줄은 하나의 메시지를 의미합니다.
- **필수**: 기승전결 4단계 구조, 5단계 이상 절대 금지
- 턴 감각: 전체 줄 수 대비 화자별 발화 비중을 50% 이하로 유지합니다. 예) 60줄 기준 화자당 30턴을 넘기지 않고 대부분 2연속 이하로 끊습니다.
- 클라이맥스 세분화: 긴장 고조를 여러 단계로 나누어 각 터치, 반응, 감정 변화를 촘촘히 배치합니다.
- **성관계 포함 시에도 총 50~80줄**을 유지하며 다음 비율을 지킵니다.
  * 기 (6~10줄): 만남→끌림, narration으로 장면/분위기 묘사
  * 승 (8~12줄): 키스→이동→옷 벗기기, narration으로 전환 묘사
  * **전 (28~44줄, NSFW)**: 
    - 전희+삽입 (6~10줄, narration으로 행위 묘사)
    - **본행위 (18~28줄)**: narration과 신음을 균형 있게 배치
    - 절정+사후 (6~10줄, narration으로 마무리)
  * 결 (8~12줄): 여운, 대화
  * **본행위 작성 규칙**: 
    - narration 30~40%, user 신음 50~60%, char 질문 10%로 배치
    - user 신음 2~3줄 → narration (상황 변화) → user 신음 2~3줄 반복
    - narration은 장면 묘사 (위치, 움직임, 표정, 분위기) 자유롭게
  * user 신음: **한 줄에 5~10개 조합** ("아앙! 흐으윽! 으응! 아아! 하아!") - 말줄임표 금지
  * **중요**: narration으로 장면 생동감, 신음으로 감정 표현, 둘의 균형 유지
- 결말: (사용자 감정 정리 → 클라이막스 → 클라이막스 정리) 흐름으로 여운을 남깁니다.
- 50줄 미만으로 끝났다면 추가 메시지를 이어서 작성해 요구 분량을 채웁니다.
- 작성 후 `<thinking>`에 적은 줄 수와 실제 출력이 일치하는지 마지막으로 확인합니다.

```
[narration] 
[user] 민준씨 어제 적극적이시던데요?
[char:차민준] 어제 {user}씨랑 궁합이 너무 잘 맞아서
[user] 저도 어제 너무 좋았어요
```

→ 사용자 감정 정리, 캐릭터 제안, 사용자 답변이 모두 있는 결 예시
- 작성 후 `<thinking>`에 적은 줄 수와 실제 출력이 일치하는지 마지막으로 확인합니다.
