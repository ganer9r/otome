# 요리 조합 시스템 설계 원칙

이 문서는 요리 게임의 레시피를 설계할 때 따라야 할 핵심 원칙을 정의합니다.

---

## 1. 등급 체계

### 등급 구분

| 등급  | 역할      | 설명                                              |
| ----- | --------- | ------------------------------------------------- |
| G급   | 기본 원소 | 8개 고정 (쌀, 밀, 고기, 해물, 채소, 과일, 물, 불) |
| F~E급 | 재료      | 요리를 만드는 부품 (밀가루, 육수, 소스 등)        |
| D~R급 | 요리      | 완성된 메뉴 (일부는 상위 조합 재료 역할)          |

### 등급 계산

**원칙**: "같은 등급 + 같은 등급 = 한 단계 상위 등급"

| 조합  | 결과 |
| ----- | ---- |
| G + G | F    |
| F + F | E    |
| E + E | D    |
| D + D | C    |
| C + C | B    |
| B + B | A    |
| A + A | R    |

**다른 등급 조합 불가**

- F + G = 불가
- D + E = 불가

---

## 2. 재료와 요리의 구분

### 등급별 허용

| 등급     | 허용                                              |
| -------- | ------------------------------------------------- |
| F~E급    | 순수 재료만 (밀가루, 육수, 소스, 반죽 등)         |
| D급 이상 | 요리만 (isIngredient인 경우도 완성된 요리여야 함) |

### 상위 조합 재료 (D급 이상)

C급 이상에서 `isIngredient: true`인 항목은 **완성된 요리**여야 함

- 허용: 불고기, 스테이크, 트러플파스타 (요리이면서 상위 재료)
- 금지: 와규, 오토로, 트러플 (재료로만 존재)

---

## 3. 재료 세분화 금지

### 원칙

"같은 재료의 부위/종류/등급 구분 금지"

### 금지

| 유형 | 예시                                   | 이유              |
| ---- | -------------------------------------- | ----------------- |
| 부위 | 오토로, 안심, 등심                     | 참치, 고기와 같음 |
| 종류 | 와규, 이베리코, 블루핀참치, 알바트러플 | 기본 재료와 같음  |
| 등급 | 블루랍스터, 황제전복, 특선회           | 상급 수식어       |

### 허용

| 유형        | 예시                             | 이유           |
| ----------- | -------------------------------- | -------------- |
| 다른 동물   | 고기, 돼지고기, 닭, 오리         | 서로 다른 재료 |
| 부위+조리법 | 삼겹살구이, 갈비찜, 안심스테이크 | 완성된 요리    |

## 4. 네이밍 규칙

### 금지 수식어

| 유형      | 예시                                     |
| --------- | ---------------------------------------- |
| 품질      | 좋은, 상급, 특제, 특선, 최고급, 프리미엄 |
| 권위      | 황제, 황실, 궁중, 로얄                   |
| 상급 의미 | 블루, 골든, 알바, 벨루가                 |

### 코스/정식 중복 금지

같은 개념의 단계별 업그레이드 금지:

- 정식 → 한정식 → 코스요리 → 미슐랭코스 (금지)

추상적 코스명 금지:

- 시그니처코스, 프렌치코스, 오마카세코스 (금지)

### 허용

- 실제 요리명: 비프웰링턴, 코키유생자크, 부야베스
- 구체적 코스 (각 카테고리 1개만): 한정식, 오마카세, 가이세키

---

## 5. 조합 규칙

### 조리도구

| 도구     | 포함 요소 |
| -------- | --------- |
| 냄비     | 물        |
| 후라이팬 | 기름      |
| 오븐     | 열만      |

### 허용/금지 조합

| 조합                      | 허용 |
| ------------------------- | ---- |
| 다른 재료 2개 + 도구      | 허용 |
| 재료 1개 + 도구           | 허용 |
| 같은 재료 2개 + 도구 없음 | 허용 |
| 같은 재료 2개 + 도구      | 금지 |

### 재료 중복 금지

직접 중복뿐만 아니라 간접 포함도 중복으로 판단

- 금지: 닭 + 닭육수 (닭육수는 닭으로 만들어짐)
- 허용: 닭 + 사골육수 (다른 재료 기반)
