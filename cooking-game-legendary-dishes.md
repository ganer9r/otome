# 🌟 요리 게임 전설의 요리 가이드

## 📊 등급 시스템

**전체 등급**: G → F → E → D → C → B → A → R → SR → SSR

**고급 요리 구성** (총 480개 중):
- **SSR급**: 5개 (1%)
- **SR급**: 12개 (2.5%)
- **R급**: 30개 (6.25%)
- **합계**: 47개 (약 10%)

---

## 🎯 설계 원칙

### 조합 구조
```
SSR급 = SR급 + SR급
SR급 = R급 + R급
R급 = A급 이하 요리 조합
```

### 네이밍 철학 (B급 게임 감성)
1. **구체적 + 과장**: "108층", "999개" 등 숫자로 과장
2. **신화적 요소**: 용, 봉황, 유니콘, 악마, 천사
3. **색감 강조**: 무지개, 검은, 붉은, 은빛, 황금
4. **웃긴데 멋있음**: "ㅋㅋㅋ 이게 뭐야" → "근데 예쁘네" → 갖고 싶음

### 비주얼 요구사항
- **정적 이미지만** 사용 (애니메이션 없음)
- **일러스트 한 장**으로 임팩트 있어야 함
- 색감이 화려하거나 독특함 (검은색, 금색, 무지개, 은빛)
- 모양이 특이함 (층, 소용돌이, 반짝임)

## 🌟 SSR급 요리 (5개)

### 최종 후보 리스트

#### 확정 후보
1. **999개 맛의 폭발 볶음밥**
   - 컨셉: 은하철도 999 패러디
   - 비주얼: 여러 재료가 박힌 황금 볶음밥, 빛나는 효과
   - 특징: 숫자 과장형

2. **108층 무지개 케이크**
   - 컨셉: 불가능에 도전하는 초고층 케이크
   - 비주얼: 하늘까지 닿는 7색 층 케이크
   - 특징: 숫자 + 색깔 과장형

3. **빛을 삼킨 검은 파스타**
   - 컨셉: 먹물 파스타의 극한 버전
   - 비주얼: 검은 면에 별처럼 반짝이는 효과
   - 특징: 색깔 극단형

4. **봉황이 품은 불꽃 통닭**
   - 컨셉: 전설의 새와 현대 요리의 만남
   - 비주얼: 날개 펼친 황금빛 통닭, 붉은 불꽃 효과
   - 특징: 신화 생물형

5. **별빛 은하수 냉면**
   - 컨셉: 우주를 담은 시원한 면 요리
   - 비주얼: 은빛 면 + 파란 육수 + 별가루 반짝임
   - 특징: 우주 컨셉형

#### 추가 후보
- **하늘을 품은 푸른 수프** (파란 나비콩 수프)
- **용암을 담은 붉은 떡볶이** (붉은 용암 연출)
- **유니콘의 무지개 꿈 젤리** (7색 층 젤리)
- **용의 눈물이 담긴 보석 탕** (맑은 육수 + 빛나는 건더기)
- **악마도 포기한 매운 떡볶이** (극한의 매운맛)

### 조합 예시 (추후 확정)
```
SSR: 999개 맛의 폭발 볶음밥
├─ SR: (미정)
│  ├─ R: (미정)
│  └─ R: (미정)
└─ SR: (미정)
   ├─ R: (미정)
   └─ R: (미정)
```

## ⭐ SR급 요리 (12개)

### 설계 방향
- R급 2개 조합으로 생성
- SSR급보다 한 단계 낮은 전설성
- 카테고리별 고르게 분배

### 네이밍 패턴 참고
- **신화 생물**: 용의 만찬, 봉황의 축복
- **마법/신비**: 마법사의 실험, 요괴의 잔치
- **대립 구조**: 악마의 유혹 vs 천사의 축복
- **극한 표현**: 시간이 멈춘, 차원을 넘은

### 카테고리 배분 (예시)
- 한식: 2개
- 양식: 2개
- 중식: 2개
- 일식: 2개
- 퓨전: 2개
- 디저트: 2개

## 🔥 R급 요리 (30개) ✅ 확정

### 설계 방향
- **A급 + A급 = R급** (기본 원칙)
- 실제 존재하는 최고급 요리 (미슐랭급)
- 게임의 핵심 매력 요리
- SR급 조합의 기초가 됨

### 조합 가능성
- 기본 재료 40개로 조합 가능
- 송로버섯 = `버섯 + 버섯` (특수 재료)
- 각 요리는 G급부터 R급까지 조합 트리 구축 가능

---

### 한식 (5개)

1. **신선로**
   - 궁중 전골의 정점, 화로에 끓이는 육해공 조합
   - 조선 왕실 연회 최고급 요리
   - 재료: 소고기, 전복, 새우, 버섯, 시금치, 계란, 당근 등

2. **구절판**
   - 9칸에 9가지 재료 배치, 색색깔 비주얼
   - 궁중 잔치 음식, 격식의 정점
   - 재료: 소고기, 버섯, 시금치, 계란, 당근 등

3. **장흥삼합**
   - 한우 + 조개 + 버섯 (육해산 조합)
   - 장흥 지역 특산물 3종 조합의 정수
   - 재료: 소고기, 조개, 버섯

4. **삼계해물탕**
   - 닭 + 새우 + 전복 (육해공 보양탕)
   - 삼계탕과 해물탕의 최고급 조합
   - 재료: 닭, 새우, 전복

5. **어복쟁반**
   - 전복 + 조개 + 오징어 냉채
   - 조선 3대 요리, 해산물 냉채의 정점
   - 재료: 전복, 조개, 오징어

### 양식 (5개)

6. **비프 웰링턴**
   - 소고기 + 송로버섯 + 파이
   - 영국/프랑스 최고급 요리
   - 재료: 소고기, 송로버섯, 밀

7. **트러플 카르보나라**
   - 송로버섯 + 계란 + 돼지고기 파스타
   - 카르보나라의 최고급 버전
   - 재료: 송로버섯, 계란, 돼지고기, 밀

8. **코키유 생 자크**
   - 조개 + 크림 + 버터
   - 프랑스 고급 레스토랑 필수
   - 재료: 조개, 우유, 버터

9. **부야베스**
   - 생선 + 해산물 스튜
   - 프랑스 3대 요리 중 하나
   - 재료: 연어, 새우, 조개, 오징어

10. **부르기뇽**
    - 소고기 + 와인 + 버섯 스튜
    - 프랑스 전통 고급 요리
    - 재료: 소고기, 포도(와인), 버섯, 양파

14. **청초해분**

15. **좌종당계**

## 🔗 조합 시스템

### 조합 규칙
1. **최대 2개 재료** 조합
2. **재료 + 조리도구** 가능
3. **완성 요리 = 새 재료** (재귀적 시스템)
4. **등급 자동 상승**: 조합 깊이에 따라 자동 결정

### 조리도구
- **냄비**: 끓이기, 찌기
- **후라이팬**: 볶기, 튀기기 (기름 포함)
- **오븐**: 굽기
